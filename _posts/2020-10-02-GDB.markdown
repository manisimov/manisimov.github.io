Load symbols from file (mess):
    > file mess
    Reading symbols from mess...done.
    > target exec mess
    > b main
    > r
    
    
Renesas simulator:
    > file rx52n.elf
    > target sim
    > load
    > b main
    > r
    
Python (https://sourceware.org/gdb/onlinedocs/gdb/Python-API.html#Python-API):
    > source mess_gdb.py
    
Pause running process:
    > [CTRL+C]
    
TUI:
    > [CTRL+x+a]
    > layout asm
